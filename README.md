# yxsc pc（云星搜索引擎PC版）
yxsc pc 是一款轻量化、纯前端实现的自定义搜索引擎主页，主打简洁纯白的界面风格与丰富的个性化功能，无需后端支持即可独立运行。

## 🌟 核心特性
### 1. 极简界面设计
- 纯白基底，支持**浅色/深色模式**一键切换，切换状态自动本地保存
- 居中布局的纯文字 Logo + 一体化搜索框，视觉简洁无冗余
- 响应式设计，适配主流PC端浏览器

### 2. 多搜索引擎支持
- 默认引擎：必应（国内版）
- 可选引擎：必应国际版、百度、360搜索
- 纯网址模式：输入网址可直接跳转，自动补全 http/https 前缀
- 引擎选择采用下拉列表形式，节省页面空间，交互更友好

### 3. 自定义捷径功能
- 禁用页面右键菜单，右键点击自动打开「捷径窗口」
- 内置热门网站捷径（B站、知乎、微博、淘宝、京东、网易云等）
- 支持自定义添加捷径，添加后自动保存到本地，刷新不丢失

### 4. 个性化设置
- **壁纸自定义**：支持上传本地图片作为背景壁纸，Base64 格式本地存储
- **设置帮助指南**：提供 Chrome/Edge/Firefox/360 浏览器主页设置教程
- **开源仓库跳转**：右上角 GitHub 图标可跳转至项目开源仓库（可自定义链接）

## 🚀 技术特点
- **纯前端实现**：基于 HTML + CSS + JavaScript 开发，无后端依赖
- **本地存储**：所有个性化配置（捷径、主题、壁纸）均通过 localStorage 保存
- **无第三方依赖**：仅引入 Font Awesome 图标库，轻量无冗余
- **开箱即用**：保存为 HTML 文件即可运行，支持部署到任意静态托管平台（GitHub Pages/Netlify 等）

## 📱 使用说明
### 基础操作
1. 在搜索框输入内容，选择对应搜索引擎后点击「搜索」或按回车键执行搜索
2. 右上角图标功能：
   - 🌙/☀️：切换浅色/深色模式
   - ❓：打开设置帮助窗口（浏览器主页设置教程）
   - ⚙️：打开设置窗口（上传/重置壁纸）
   - 🐙：跳转至 GitHub 开源仓库
3. 右键点击页面任意位置：打开捷径窗口，点击捷径可直接跳转对应网站
4. 捷径窗口支持输入「网站名称+网址」自定义添加新捷径

### 个性化配置
- **壁纸设置**：在设置窗口点击「选择壁纸」上传本地图片，或点击「重置壁纸」恢复默认
- **主页设置**：在「设置帮助」窗口查看各浏览器设置主页的详细步骤
